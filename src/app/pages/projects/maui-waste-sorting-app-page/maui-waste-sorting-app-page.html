Projectoverzicht

Een app die gebruikmaakt van Azure Computer Vision om gebruikers te helpen hun afval correct te sorteren. Gebruikers kunnen items scannen (bijv. plastic, glas of organisch afval), waarna de app het type afval herkent en aangeeft hoe je het correct moet weggooien.

Appflow

  • Foto → Azure AI Custom Vision model
  • Resultaat → vragen aan API (bijv. zak chips → API stuurt terug "PMD")
  • Gebruiker bevestigt of het resultaat correct is in de app
  • Bij bevestiging wordt dit in de API geregistreerd

Userflow

  • Scan afval
  • De app geeft aan wat het is en in welke zak het moet
  • Voorbeelden:
    • Glazen fles → glasbak
    • Blik → PMD
    • Drankkarton → PMD
  • Je bevestigt of de herkenning klopt
  • In de API wordt het item geteld
  • Op een aparte pagina zie je hoeveel van elk item in elke zak zit
  • Voorbeeld:
    • In mijn PMD-zak zitten: 5 blikken, 3 zakken chips, 7 plastics, 2 drankkartons
    • In mijn glasbak zitten: 5 glazen flessen
  • Weergave per afvalzak is mogelijk
